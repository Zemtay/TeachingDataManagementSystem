# 智能学生课程管理系统 TeachingDataManagementSystem

本项目聚焦教育数字化转型需求，构建基于**多维度教学数据分析的智能辅助系统**，底层依托 RuoYi 框架实现 Spring Boot (Java) + Vue 前后端分离架构，融合 Django 生态的数据分析能力，面向高校、中小学及培训机构的教学管理场景，提供教学数据智能解析、多维度可视化呈现、自然语言交互分析三大核心能力，创新打造教师/管理员双视角分析体系，辅助学情诊断与教学资源优化配置。

---

## 功能概览

- **数据管理** ：支持Excel/CSV格式教学数据上传、自动化清洗（去重/格式标准化）、结构化存储至MySQL数据库
- **课程与班级管理** ：课程/班级信息创建、批量编辑、关联关系维护，支持按学期/专业/教师维度筛选统计
- **教学数据分析** ：
  - 教师视角：学生任务点完成、课堂互动（讨论/发帖/回帖）、成绩分布（箱形图/雷达图）、学习活跃度分析
  - 管理员视角：教师授课量、课程开设规模、教学资源分布、跨班级/跨课程全局统计
- **数据可视化** ：基于Matplotlib（静态报告）+ chart.js（动态交互）双引擎，生成柱状图/折线图/散点图/雷达图等多类型图表，支持自定义数据看板
- **AI智能助手** ：集成GPT-4o-mini大模型，通过自然语言意图识别+Text2SQL技术，实现对话式数据查询，自动生成教学改进智能诊断报告
- **用户与权限** ：精细化角色隔离（教师/管理员），基于工号匹配+行级权限过滤，保障数据隐私；PBKDF2算法加密存储用户口令，防爆破/彩虹表攻击
- **系统日志** ：操作日志/登录日志全记录，支持按操作人/类型/时间筛选，满足教学管理审计需求

---

## 技术架构

- **核心框架** ：Spring Boot (Java) + RuoYi（权限/安全）、Django（数据分析/ORM）
- **前端技术** ：Vue、Element UI、chart.js（动态可视化）、AJAX（异步交互）
- **数据分析** ：Python Pandas（数据清洗）、Matplotlib（静态可视化）、GPT-4o-mini API（自然语言交互）
- **数据库** ：MySQL（核心教学数据存储，支持PostgreSQL小适配）
- **安全机制** ：PBKDF2口令加密、路由守卫、行级权限过滤、会话管理
- **接口风格** ：RESTful，支持批量操作、分页、复杂条件检索，全接口详见 [API文档.md](API文档.md)

## 快速启动

1. **环境准备**
    - 基础环境：JDK 8+/11+、Node.js 14+、MySQL 5.7+/8.0+
    - 数据分析环境：Python 3.8+（Pandas/Matplotlib）、Django 4.0+
2. **后端部署**
    - 配置数据库连接（`application.yml`/Django `settings.py`）
    - 初始化核心数据表（CourseInfo/ClassInfo/Student/CourseActivity）
    - 启动 Spring Boot 服务（`ruoyi-system`/`ruoyi-admin` 模块）+ Django 数据分析服务
3. **前端部署**
    - 进入 `ruoyi-ui` 目录执行 `npm install && npm run dev`
    - 前端默认端口：http://localhost:8080（可自定义）
4. **数据初始化**
    - 管理员登录后上传Excel/CSV格式教学数据（含课程/班级/学生/互动数据）
    - 系统自动完成数据清洗与结构化存储
5. **访问系统**
    - 默认入口地址：http://localhost:8080（按实际配置端口调整）

## 适用场景

- **教师** ：
  - 学情诊断（学生任务完成率、互动活跃度、成绩离散度分析）
  - 教学策略调整（基于AI建议优化课堂互动、任务布置）
  - 班级/课程数据可视化查看与导出
- **管理员/教务** ：
  - 角色权限配置、教学数据统一管理与清洗
  - 全局教学资源监控（教师授课量、课程开设规模）
  - 生成校级/机构级教学数据智能诊断报告，优化资源配置
- 支持多院系、多岗位角色授权，适配高校/培训机构个性化管理需求

## 项目亮点

- **双架构融合** ：Spring Boot+RuoYi保障权限与业务稳定性，Django提升数据分析灵活性，兼顾易用性与拓展性
- **双视角分析体系** ：教师聚焦学情细节，管理员掌控全局资源，形成教学数据闭环
- **自然语言交互** ：基于Text2SQL与意图识别技术，非技术用户可通过日常语言查询复杂教学数据
- **多维度可视化** ：静态（Matplotlib）+ 动态（chart.js）双引擎，支持成绩分布、互动趋势、学情均衡性等多维度图表展示
- **数据安全保障** ：PBKDF2加密、行级权限过滤、操作日志审计，全面保护教学敏感数据
- **智能诊断建议** ：AI助手基于教学数据自动生成学情预警、资源配置优化建议，助力数据驱动决策
- **易扩展特性** ：支持批量操作、复杂条件检索，可快速适配新的教学数据维度（如作业/考试/投票数据）
